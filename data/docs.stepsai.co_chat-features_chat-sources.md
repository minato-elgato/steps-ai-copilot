## Getting Started

### 1\. Selecting Your Sources

Access your preferred data sources directly from the search/input bar:

- Web Search
- Google Drive
- Slack
- And more…

:::tip
**Important**: Make sure each source shows a “Connected” status before proceeding. Look for the green connection indicator next to each source.
:::

### 2\. Managing Sources

Once selected, your chosen sources will appear as labels in the chat interface. This allows you to:

- Easily track which sources are active
- Remove sources when no longer needed
- Add additional sources during the conversation

### 3\. Start Chatting

With your sources connected, you can:

- Ask questions across multiple sources simultaneously
- Compare information from different platforms
- Get comprehensive answers drawing from all connected sources

## Tips for Effective Source Management

- Connect only the sources relevant to your current conversation
- Verify source connections before starting your chat
- Use source labels to track where information is coming from
- Add or remove sources as your conversation evolves
