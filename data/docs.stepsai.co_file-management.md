## Step-by-Step Guide

1. **Add Files**:   - Click the attachment icon in chat
   - Select files from your device
   - Supported formats:
     - PDFs
     - Images
     - Documents
     - Text files
2. **Manage Attachments**:   - View attached files in thread
   - Download files as needed
   - Remove attachments if required
